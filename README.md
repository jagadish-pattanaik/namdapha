🚀 Quick Start
# Clone the repository
git clone https://github.com/yourusername/namdapha.git

# Navigate to the project directory
cd namdapha

# Install dependencies
npm install

# Start development server
npm run dev
Visit http://localhost:3000 to see the application!

🏗️ Project Structure
namdapha/
├── app/                  # Next.js App Router pages
│   ├── about/            # About page
│   ├── events/           # Events calendar
│   ├── house-council/    # House Council information
│   ├── imp-links/        # Important Links page
│   ├── resource-hub/     # Resource Hub 
│   ├── teams/            # Teams directory
│   ├── layout.tsx        # Root layout with background
│   └── page.tsx          # Home/landing page
├── components/           # Reusable UI components
├── public/               # Static assets
└── styles/               # Global styles

🧩 Key Components
Glassmorphism Cards: Modern transparent UI elements with backdrop blur
Dynamic Resource Filtering: Category and search-based filtering system
Responsive Navigation: Adaptive navbar with mobile menu
Theme Toggle: Switch between dark and light modes
Internal Routing: Seamless navigation between application pages

📋 Migration Notes
This project was migrated from a React SPA to Next.js App Router architecture for improved performance and developer experience.

Key migration changes:

React Router → Next.js App Router
Component adaptation for Server/Client rendering
Image optimization with Next.js Image component
Enhanced styling with Tailwind CSS v4
